Details :

This is a simple web app which can be used as a "captcha" validator before entering any websites, and this ensures that the traffic your site experience is caused by humans and not by bots. This Captcha Validator has a 7- alphanumeric character set which cannot be By-passed using Bots. This is made with the help of HTML, CSS and JAVASCRIPT.
